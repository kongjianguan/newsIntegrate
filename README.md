# News Aggregator — 多源新闻聚合工具

环境工程领域发展迅速，新技术、新政策、新研究层出不穷。每天浏览多个学术期刊、新闻站点和行业博客以获取最新动态，耗时且容易遗漏。

本工具集成多源信息，自动完成收集、整理、分类，让你在一个地方就能掌握全领域动态。

## 功能

- 多信源集成 —— 聚合多个环境工程相关信源，一站式浏览
- 智能分类 —— 内容自动归类，快速筛选感兴趣的方向
- 定时更新 —— 按设定的频率自动获取最新内容
- 关键词搜索 —— 在海量信息中快速定位目标
- 支持自定义LLM提供商

## 开始前准备（以DeepSeek为例）

- 在run.py同级目录下新建.env文件
- 填入API key，如
```
DEEPSEEK_API_KEY=sk-*****************
```

## 快速开始

```bash
# 安装依赖
pip install -r requirements.txt

# 启动
python run.py
```

启动后浏览器打开 `http://localhost:8765` 即可使用。
